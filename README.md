# Компьютерное моделирование
# Лабораторная работа 1

### Задание
1. Решить задачу коммивояжера одной из метаэвристик:
- имитация отжига;
- генетический алгоритм;
- эволюционные стратегии;
- алгоритм муравьиной колонии.
2. Выполнить численный эксперимент на тестовых наборах данных из международной библиотеки тестов "OR-library".
3. Оформить один отчёт на группу и таблицу, в которой результаты экспериментов на одних и тех же тестовых данных для всех алгоритмов сводятся в одну таблицу.
